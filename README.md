# Curso-Django
repositório para estudo do framework Django

# Inicar o Porjeto
1.  [powershell] Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
2.  [powershell] .\.venv\Scripts\activate
3.  [powershell] .\manage.py runserver

# Migrações 
1.  [powershell]  python manage.py makemigrations
2.  [powershell] python manage.py migrate
